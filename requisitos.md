# BookHub – Requisitos Funcionais

## 1. Módulo de Autenticação e Conta

- RF-01 – O sistema deve permitir o cadastro de novos usuários com:
  - Nome
  - Email único
  - Senha criptografada

- RF-02 – O sistema deve permitir login com email e senha.

- RF-03 – O sistema deve permitir logout do usuário autenticado.

- RF-04 – O sistema deve permitir recuperação de senha via email.

- RF-05 – O sistema deve permitir edição de dados do perfil:
  - Nome
  - Email
  - Senha

- RF-06 – O sistema deve permitir exclusão da conta do usuário.

---

## 2. Módulo de Exploração de Livros

- RF-07 – O sistema deve listar livros disponíveis na plataforma.

- RF-08 – O sistema deve permitir navegação por categorias.

- RF-09 – O sistema deve permitir busca de livros por:
  - Título
  - Autor
  - Categoria

- RF-10 – O sistema deve exibir detalhes do livro:
  - Título
  - Autor
  - Descrição
  - Categoria
  - Avaliação média
  - Número de avaliações

---

## 3. Módulo de Leitura Online

- RF-11 – O sistema deve permitir leitura online de livros.

- RF-12 – O sistema deve exibir o conteúdo do livro por páginas ou capítulos.

- RF-13 – O sistema deve permitir navegação entre páginas/capítulos.

- RF-14 – O sistema deve salvar automaticamente o progresso de leitura.

- RF-15 – O sistema deve permitir continuar a leitura de onde o usuário parou.

---

## 4. Módulo de Favoritos

- RF-16 – O sistema deve permitir adicionar livros aos favoritos.

- RF-17 – O sistema deve permitir remover livros dos favoritos.

- RF-18 – O sistema deve exibir a lista de livros favoritos do usuário.

---

## 5. Módulo de Histórico de Leitura

- RF-19 – O sistema deve registrar livros acessados pelo usuário.

- RF-20 – O sistema deve exibir histórico de leitura.

- RF-21 – O sistema deve exibir progresso de leitura por livro (ex: % ou capítulo atual).

- RF-22 – O sistema deve permitir limpar o histórico.

---

## 6. Módulo de Avaliações e Comentários

- RF-23 – O sistema deve permitir que usuários avaliem livros (ex: 1 a 5 estrelas).

- RF-24 – O sistema deve permitir adicionar comentários aos livros.

- RF-25 – O sistema deve permitir editar comentários próprios.

- RF-26 – O sistema deve permitir excluir comentários próprios.

- RF-27 – O sistema deve exibir avaliações e comentários de outros usuários.

- RF-28 – O sistema deve calcular e exibir média de avaliações por livro.

---

## 7. Módulo Administrativo – Dashboard

- RF-29 – O sistema deve exibir total de usuários cadastrados.

- RF-30 – O sistema deve exibir total de livros cadastrados.

- RF-31 – O sistema deve exibir estatísticas de leitura:
  - Livros mais lidos
  - Usuários mais ativos

---

## 8. Módulo Administrativo – Gerenciamento de Livros

- RF-32 – O administrador deve poder cadastrar novos livros.

- RF-33 – O administrador deve poder editar livros existentes.

- RF-34 – O administrador deve poder excluir livros.

- RF-35 – O administrador deve associar livros a categorias.

- RF-36 – O administrador deve fazer upload de conteúdo do livro (PDF, texto, etc.).

---

## 9. Módulo Administrativo – Categorias

- RF-37 – O administrador deve poder criar categorias.

- RF-38 – O administrador deve poder editar categorias.

- RF-39 – O administrador deve poder excluir categorias.

- RF-40 – O sistema deve associar livros a uma ou mais categorias.

---

## 10. Módulo Administrativo – Usuários

- RF-41 – O administrador deve visualizar lista de usuários.

- RF-42 – O administrador deve visualizar detalhes de um usuário.

- RF-43 – O administrador deve bloquear/desbloquear usuários.

- RF-44 – O administrador deve excluir contas de usuários.

- RF-45 – O administrador deve definir permissões (usuário/admin).

---

## 11. Módulo de Notificações (Opcional)

- RF-46 – O sistema deve notificar usuários sobre:
  - Novos livros
  - Respostas a comentários

- RF-47 – O sistema deve permitir visualizar notificações.
